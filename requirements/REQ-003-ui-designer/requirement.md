# REQ-003: UI Designer — Side Panel Improvements

## Summary

Improve the side panel chat UI for better usability and visual design.

## Requirements

1. **推挤布局** — 侧面栏打开时不遮挡页面内容，而是把页面内容挤到左边（缩小 body 宽度）
2. **固定宽度** — 侧面栏宽度固定 380px
3. **关闭按钮** — 点击后侧面栏收回，页面恢复原始宽度
4. **清除历史按钮** — header 区域增加清除按钮，点击后清空聊天记录，显示默认提示信息
5. **背景色** — 面板背景浅灰色(#f9fafb)，区分于主内容区域
6. **字体** — 系统默认字体，14px，黑色
7. **标题区域** — 深灰色背景(#374151)，白色字体，显示 "ChromePilot"
8. **内边距** — 内容区域 20px 内边距
9. **滚动** — 内容超出时可滚动
10. **动画** — 弹出和收回平滑过渡，300ms
11. **响应式** — 小屏幕下面板宽度自适应（不超过屏幕 80%）
12. **输入框兼容 Vim 插件** — 输入框必须是真正的 `<input>` 或 `<textarea>`，且在 focus 时阻止事件冒泡到页面（防止 Vim
    插件拦截按键）
13. **默认提示** — 聊天区域为空时显示欢迎提示信息

## Acceptance Criteria

- [ ] 打开面板时页面内容被挤到左边，不被遮挡
- [ ] 关闭面板时页面恢复原始宽度
- [ ] 清除按钮清空聊天记录并显示默认提示
- [ ] 面板有平滑的弹出/收回动画（300ms）
- [ ] 输入框在 Vim 插件环境下正常输入不冲突
- [ ] 小屏幕下面板宽度不超过屏幕 80%
- [ ] 面板背景、标题栏、内边距符合设计规范
